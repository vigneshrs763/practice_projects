# 🚀 JavaScript Practice Projects

A collection of beginner-friendly JavaScript projects built to improve my programming and problem-solving skills.

## 📌 Projects Included

### 🌡️ Temperature Converter
Convert temperatures between:
- Celsius (°C)
- Fahrenheit (°F)
- Kelvin (K)

**Features**
- Instant conversion
- User-friendly interface
- Input validation

---

### 🧮 Calculator
A simple calculator for performing basic arithmetic operations.

**Features**
- Addition
- Subtraction
- Multiplication
- Division
- Clear/Delete functionality

---

### 🎲 Dice Roller Game
A random dice rolling simulator built using JavaScript.

**Features**
- Roll single or multiple dice
- Random number generation
- Interactive UI
- Instant results display

---
